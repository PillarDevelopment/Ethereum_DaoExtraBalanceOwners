// WHAT's SPECIAL ABOUT THESE 140 addresses?
// they are all smart contracts.

// WHY DO WE NEED THEM
// these smart contracts have a function that calls theDAO createToken function
// but if you simply look at them in the simple transaction   web3.eth.getTransaction(txHash)
// the DAO won't appear in the TO field, but rather, you will see the address of the smart contract
// the only way to tell that these transactions are in fact to theDAO is by tracing them
// and looking in the stack of their CALL functions if there are DAO related references (function hashes, dao Address and dao ExtraBalance address)

// the appropriate thing to do is to trace ALL transactions in the extraBalance period , but that would take a very long time
// (without this step it already took almost 3 days to parse only transactions that were obviously To theDAO)
// therefore what we will do is simply:
//      - parse all blocks and transactions in the extraBalance period
//      - if we find a transaction whose TO is one of these 140 known smart cotract addresses, we will parse it as if it was a normal transaction to theDAO
